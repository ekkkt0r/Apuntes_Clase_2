## Arquitectura del pensamiento  
- Máquina diferencial: Babbage diseño esta máquina para sistematizar el cálculo y tabulación de funciones polinómicas. Utilizaba diferencias finitas para calcular valores complejos mediante sumas, evitando multiplicaciones o divisiones complejas.
- Telar de Jacquard: inventado por *Joseph Marie Jacquard*, fue el primer sistema de almacenamiento de información binaria. permitía automatizar la creación de patrones en los tejidos mediante tarjetas perforadas. Sistema considerado como antecedente importante de la programación y la computación.
Este telar introduce 3 conceptos fundamentales de la computación moderna:
1. La separación entre hardware y software
2. El sistema binario
3. El pixel
